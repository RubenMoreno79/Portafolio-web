SobreMí

Full Stack Developer

Soy diseñador gráfico con experiencia y recientemente he dado un giro en mi carrera profesional y me he formado como desarrollador Full Stack. Ahora combino mis conocimientos de diseño con mis habilidades de programación para crear soluciones integrales, tanto visuales como funcionales.

Mi enfoque abarca desde el front-end, utilizando tecnologías como HTML, CSS, JavaScript, Angular, Figma y WordPress, hasta el back-end con Node.js, Express.js y bases de datos como MongoDB y MySQL. Me apasiona crear interfaces intuitivas y amigables que no solo luzcan bien, sino que también ofrezcan una experiencia de usuario fluida y eficiente.

Mi objetivo es seguir creciendo como desarrollador mientras aplico mis conocimientos para crear productos digitales que impacten de manera positiva.

Mis Servicios

Front-End

Combino habilidades técnicas y creativas para crear interfaces de usuario visualmente atractivas y funcionales. Trabajo con tecnologías como HTML5, CSS3, JavaScript y Angular 17. Me gusta prestar especial atención a los detalles visuales, asegurando que cada proyecto no solo sea responsive y accesible, sino también estéticamente coherente. Aunque estoy en los primeros pasos de mi carrera en desarrollo web, tengo una base sólida para crear experiencias de usuario efectivas y atractivas.

Back-End

En mi rol de desarrollador Back-End me enfoco en construir y mantener la lógica de las aplicaciones para asegurar su correcto funcionamiento. He trabajado con tecnologías como Node.js, Express.js, y tengo experiencia con bases de datos relacionales y no relacionales como MySQL y MongoDB. Me apasiona aprender y mejorar mis habilidades en la integración de APIs, autenticación de usuarios y gestión de datos. Siempre busco optimizar el rendimiento y la seguridad del servidor.

Diseño UX/UI

Una de mis mayores pasiones es el diseño UX/UI, mi objetivo es crear experiencias de usuario intuitivas, atractivas y funcionales. He estudiado la metodología Design Thinking y he aprendido a gestionar los proyectos gracias al marco de gestión de Scrum y Kanban. Además de las herramientas que utilizo para el front, poseo conocimientos en Figma, Wordpress con PHP y Elementor para desarrollar prototipos y diseños visuales que se ajusten a las necesidades de los usuarios y los objetivos del negocio de manera responsive.

Mis Proyectos

Portfolio

Diseñado para destacar mis habilidades como programador full stack. El sitio presenta mis proyectos, experiencia y conocimientos visualmente atractiva, con una interfaz interactiva y optimizada para una navegación responsive para móvil. Proyecto hecho en HTML, CSS y JavaScript.

Makeup Store

Tienda online de maquillaje. El sitio cuenta con un diseño moderno, optimizado para dispositivos móviles y fácil de navegar. Incluye una pasarela de pagos segura, carro de compra, productos destacados y mucho más. Proyecto hecho en WordPress utilizando Elementor.

Relax Spa

Comercio de centro de masajes, ofrece información clara sobre servicios, precios y reservas. Está optimizada para dispositivos móviles y proporciona una experiencia de usuario intuitiva y eficiente para los clientes. Proyecto hecho en WordPress utilizando Elementor

Colegio Wordpress

Incluye secciones de información, noticias, eventos escolares, calendario y etiquetas. La página está organizada para facilitar la navegación de estudiantes, padres y profesores. Creada con temas, pluglins y es responsive para todos los dispositivos. Proyecto hecho en WordPress.
